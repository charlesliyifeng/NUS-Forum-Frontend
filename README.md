## Getting Started

### Running the app

1. Install dependencies for the project by entering this command:

```bash
yarn install
```

2. Run the backend

3. Run the app in development mode by entering this command:

```bash
yarn start
```

4. Open [http://localhost:3001](http://localhost:3001) to view it in the browser.
